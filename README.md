# Employee Manager System 👨‍💼

一个基于 **Python** 编写的命令行员工管理系统，支持用户注册、登录以及管理员权限管理等功能。

---
## 📌 项目说明

这是一个用于练习 GitHub 上传流程的项目

---

## 🚀 项目简介

本项目是一个简单的员工信息管理系统，主要用于练习 Python 基础语法、数据结构以及简单的权限控制逻辑。

适合：

* Python 初学者练习项目
* Git / GitHub 项目上传练习
* 命令行程序开发入门

---

## ✨ 功能列表

* 🔐 用户注册 / 登录
* 👤 权限管理（管理员 / 普通用户）
* ➕ 添加用户（仅管理员）
* 📋 查看所有用户
* 🔍 查看指定用户
* ❌ 删除用户
* 🧹 清空所有用户
* ✏️ 修改用户信息

---

## 🛠 技术栈

* Python 3.x
* 标准库（无第三方依赖）

## 🔐 权限说明

| 角色         | 权限          |
| ---------- | ----------- |
| 管理员（admin） | 可添加、删除、修改用户 |
| 普通用户（user） | 仅可查看        |

* 使用字典结构管理用户数据
